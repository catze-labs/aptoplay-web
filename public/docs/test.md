# AptoPlay Installation

## Description

aptoplay-core is TS library for simplify gaming development with Aptos Chain and PlayFab

## Pre-Requirements

- NodeJS >= 16
- TypeScript >= 4.x

## How to use

```bash
$ npm install aptoplay-core

# or

$ yarn add aptoplay-core
```
